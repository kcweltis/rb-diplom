# Русские блины — дипломный проект (скелет)

Стек: Node.js + Express + EJS + PostgreSQL (без ORM).  
Роли: ADMIN, MODERATOR, COURIER, USER.  
Данные редактируются через админ-панель. Главная подтягивает "популярные блюда" из БД.

## 1) Установка
1. Установи Node.js LTS.
2. Установи PostgreSQL через установщик (Windows) и запомни:
   - user (например `postgres`)
   - password
   - port (обычно 5432)
3. Создай базу данных `rb_db` (через pgAdmin или psql).

## 2) Настройка окружения
Скопируй `.env.example` → `.env` и заполни `DATABASE_URL` и `SESSION_SECRET`.

## 3) Запуск
```bash
npm i
npm run db:migrate
npm run db:seed
npm run dev
```
Открой: http://localhost:3000

## 4) Вход
После seed создаются роли. Пользователя-админа создаёшь через страницу регистрации,
а затем вручную меняешь роль на ADMIN в БД (самый быстрый старт),
или добавь "создать админа" в seed (мы можем сделать позже).

## 5) Что уже есть
- Главная (динамические популярные блюда)
- Регистрация/вход (bcrypt + session)
- Админка:
  - Категории
  - Товары
  - Популярные блюда (featured)
  - Пользователи (смена роли/блокировка, сброс пароля)
